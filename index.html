Qui suis-je ?

Je suis Nimotalah Alli, developpeuse web en BUT3 parcours Developpement Web et Dispositifs Interactifs. Passionnee par la creation d'applications web completes, je m'investis autant dans la solidite du back-end que dans la qualite de l'experience utilisateur.

RJ10 Rappel

Application de rappel pour le renouvellement de certifications — Gestion des echeances professionnelles

Contexte

RJ10 Rappel est une application web concue pour aider les professionnels a suivre et renouveler leurs certifications a temps. Le projet repond a un besoin concret : ne plus rater une echeance critique.

Fonctionnalites cles

  • Tableau de bord des certifications avec statut et echeances
  • Systeme de rappels automatiques par email/notification
  • Gestion multi-utilisateurs avec roles
  • Interface intuitive et responsive

Stack technique

Noor App

Application full-stack — Back-end API REST + Front-end React

Contexte

Noor App est une application web complete construite en deux parties distinctes mais complementaires : un back-end robuste sous Node.js/Express et un front-end React soigne. Le projet integre une authentification securisee et une gestion fine des droits.

Back-end (noor-back)

  • API REST structuree avec Express
  • Authentification JWT et gestion des roles (admin/user)
  • Base de donnees PostgreSQL avec ORM
  • Middleware de validation et gestion d'erreurs

Front-end (noor-app)

  • Interface React avec gestion d'etat
  • Consommation de l'API via Axios
  • Routes proteges selon les roles
  • UI responsive et accessible

Stack technique

React Node.js Express PostgreSQL JWT Axios
