Projet Pro

RJ10 Rappel

Application web de rappel automatique pour le renouvellement de certifications professionnelles — conçue lors de mon stage chez RJ10 Formation.

Symfony 7 Vue.js 3 JWT Auth Symfony Mailer Messenger En cours

📋 Contexte

RJ10 Formation est un centre de formation en sécurité basé à Troyes. L'enjeu : automatiser les rappels de renouvellement de certifications (SST, habilitations électriques, etc.) pour les clients professionnels. J'ai conçu et développé cette application de A à Z dans le cadre de mon stage de BUT3.

🖼️

Ajoute une capture du dashboard ici

🖼️

Interface de gestion des certifications

🖼️

Email de rappel automatique

⚡ Fonctionnalités

  • Tableau de bord des certifications avec statut et échéances
  • Rappels email automatiques (Symfony Messenger + Mailer)
  • CRUD complet — ajout, édition, suppression des certifications
  • Authentification sécurisée JWT + rôles admin/utilisateur
  • Renvoi manuel d'email depuis l'interface admin
  • Flux mot de passe oublié / réinitialisation

🛠️ Stack technique

  • Back-end : Symfony 7 + Doctrine ORM
  • Front-end : Vue.js 3 (Composition API)
  • Auth : JWT via LexikJWTAuthenticationBundle
  • Emails : Symfony Mailer + Messenger (async)
  • Base de données : MySQL
  • Déploiement : VPS étudiant Apache

📐 Architecture & choix techniques

L'application suit une architecture API-first : Symfony expose une API REST consommée par Vue.js. Les rappels sont gérés via le composant Messenger de Symfony pour un traitement asynchrone fiable. Les tokens JWT garantissent une authentification stateless adaptée au front-end SPA.
